下載hexo指令整理

發表於 2020-04-02 更新於 2020-05-04 分類於 部落格建立

install

安裝輔助程式

git
Node.js
VScode即為Version Studio Code非必要，安裝可使介面較易懂。

安裝Hexo

在你的電腦打開git.bash或是命令提示字元視窗(cmd)
  • cmd為command縮寫，為Windows自帶命令視窗，打開方式:Windows>Search for>cmd
  • 若使用windows PowerShell在執行下載時會有版本不相容問題
輸入下載hexo的指令碼

小註:

  • npm: node package manerger，以JavaScript編寫，為Node.js預設的軟體套件管理系統，使用者可以上傳自己編寫的套件軟體，提供不同功能的套組下載。
  • cli: command line interface命令行介面，意即使用非圖標式的鍵盤輸入命令方式。
建立初始hexo化名為”filename”的資料夾

確認位在”filename”的檔案裡，重新安裝hexo

這樣我們就得到一個初始化的hexo安裝包，可以進行離線的部落格書寫啦。

npm install hexo-admin –save
