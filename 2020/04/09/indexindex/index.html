《Git-it指令小抄》

小提示

方向上鍵 可重複上一條指令
方向下鍵 可重複下一條指令

初始設定

$ git config --global設定區域變數(?)

git config --global user.name "yourname"設定你的名字
git config --global user.email "youremail"設定你的電郵
git config --global user.username "yourGitHubAccount"連動GitHub帳號

連動資料夾以及儲存變動Add&Commit

git init將資料夾加上git功能
git status查看檔案狀態
git add foldername準備commit一個檔案
git add .準備commit所有檔案
git diffcheck out the difference
git commit -m "yourmessage"commit(儲存)你的變更並附上簡短說明

新增分支Branch

git branch新增分支(後面直接加分支名字)
git checkout切換分支到(直接打分支名)

遠端連結Remote以及與他人協作

git clone複製專案到本地端
git remote add origin <URLFROMGITHUB>遠端控制GitHub repository
git remote set-url origin <URLFROMGITHUB>將以上網指內容新增至名為origin的remote
git push origin master 把分支master 傳送到新增的remote
git remote add upstream <URLFROMGITHUB>將分支暫存在向<想上傳變更的網址>的遠端
git remote -v檢查導向網址以及輸出網址

#####　紀錄變更上傳個人空間撇步

git pull 更新其他的使用者對分支進行的更改
